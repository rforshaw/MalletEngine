Detailed Description

DrawDelegate provides access for adding/removing Draw objects from the rendering-system.

On an Add or Remove request the Draw object is added to a queue, and processed by the rendering-system at a later time.

Draw/Camera objects are added to a World, the user can specify the World in which these object are added. If no World is specified then they are added to the default world.

DrawDelegate can also add Worlds to the rendering-system.
