static Event<AnimationDelegateCallback> com.linxonline.mallet.animation.AnimationAssist.constructAnimationDelegate ( final AnimationDelegateCallback  _callback)
static

Request an AnimDelegate from the active animation system. The AnimDelegate allows the user to add/remove Anim objects from being rendered.

An AnimDelegate is not required for constructing an Anim object, but is required for displaying it.
