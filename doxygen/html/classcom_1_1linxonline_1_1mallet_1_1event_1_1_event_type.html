Detailed Description

The aim of Event Type is to provide faster checks when processing Events. An EventType is created to match-up with a specific keyword, all Events that use that specific keyword will be given the same EventType. Instead of a String comparison being used, a memory address comparison is done. This implementation still allows the developer to handle Events as if their EventType was a String.
