Detailed Description

Interface for retrieving and setting data to be displayed to the user. Multiple views should be able to attach themselves to the same model.

Extend this class when implementing a new data-set. This can be used in conjunction with com.linxonline.mallet.io.save.state.DataState, allowing for easier loading/saving of data-sets too.
