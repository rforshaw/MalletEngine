Detailed Description

To initialise low-level/Operating specific systems that the game requires. It's responsibility entails creating a window, hooking-up the input-system, initialising the rendering system, and initialising the audio-system. It also handles the responsibility of creating O/S specific locks that the game requires.

EventSystemInterface com.linxonline.mallet.system.SystemInterface.getEventInterface ( )

The root Event-system, typically one state will be hooked. Allows the state to be informed of external events. For example: shutdown, minimise, layout change requests that the state may be interested in. It also enables the state to make O/S specific requests that only work on certain implementations. For example: Displaying virtual-keyboard, opening a web browser, etc.
