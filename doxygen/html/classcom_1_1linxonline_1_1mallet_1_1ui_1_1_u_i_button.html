Detailed Description

Used to determine if the user has acted within a UI area.

When the user has clicked, rolled over, or the location is reset to a neutral position an event defined by the developer is sent through the entity's event-system.

The event can then be picked up by other components such as a render-component to modify the visual element of the entity.
