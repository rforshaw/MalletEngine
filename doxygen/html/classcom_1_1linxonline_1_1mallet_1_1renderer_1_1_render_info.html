Detailed Description

Convenience class to handle Display & Render Dimensions. Provides scaling information between Display & Render dimensions, retaining or not Aspect Ratio. Also handles conversion for user-input between Display & Render.

final Vector2 com.linxonline.mallet.renderer.RenderInfo.getScaledRenderDimensions ( )

Return the render dimensions scaled to display dimensions. For example: if the render-buffer was 800x600 & display was 1024x768 it will return x:1024, y:768. The scale result will change depending on aspect-ratio and whether it is being enforced.
