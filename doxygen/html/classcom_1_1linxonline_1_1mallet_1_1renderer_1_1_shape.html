static Shape com.linxonline.mallet.renderer.Shape.combine ( final Shape...  _shapes)
static

Combine an array of shape objects into 1 shape object. Ensure all the shapes being combined have the same style. This is a convience function to improve rendering performance, instead of multiple draw calls per shape. Constructing multiple shapes and then combining them is slow and memory intensive.
